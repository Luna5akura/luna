## 第二章：排队论
- **系统组成**：到达（λ，M/G）、服务（μ，M/G）、服务器K、容量、纪律（FCFS）。
- **性能指标**：L_s = λW_s, L_q = λW_q (Little定律)；ρ = λ/(Kμ) <1。
- **M/M/1**：稳态p_n = (1-ρ)ρ^n；L_s = ρ/(1-ρ)；W_s = 1/(μ-λ)。
- **M/M/K**：p_0 = [∑_{n=0}^{K-1} (ρ^n/n!) + ρ^K/(K!(1-ρ/K)) ]^{-1}；L_q用Erlang公式。
- **M/G/1**：嵌入链，L_q = (λ²σ² + ρ²)/(2(1-ρ)) (Pollaczek-Khinchine, σ²服务方差)。
- **G/M/1**：类似嵌入链，求根方程。
- **优化**：min成本 = 服务成本 + 等待成本；静态控制λ。
- **分析思路**：MM型-流量平衡求p_n；MG型-嵌入马氏链；GG型-难深入，用近似。
- **启示**：进出平衡；服务不可预存（永远缺货），与库存比较五要素（服务 vs 硬件）。

## 第三章：排队网络
- **开放（Jackson）**：话务方程λ_i = λ_0 q_i + ∑λ_j p_{ji}；稳态概率乘积，各节点独立M/M。
- **密闭（Gordon-Newell）**：固定N顾客，访问率平衡，概率正比访问率。
- **混合（BCMP）**：结合开放/密闭，一般分布。
- **思路**：推广M/M/1，只求中心进出平衡，仿佛独立；子集平衡。
- **启示**：系统/子系统平衡，促进独立分析。

## 第四章：马尔可夫决策过程
- **模型**：状态S_n、动作A(i)、奖励r(i,a)、转移p_{ij}(a)、目标V(π,i)。
- **有限阶段**：V_n(i) = max_a [r + ∑ p_{ij} V_{n-1}(j)] (Bellman)。
- **无限折扣（β<1）**：V(i) = max_a [r + β∑ p V(j)]；算法：价值/策略迭代、LP。
- **平均准则**：g + h(i) = max_a [r + ∑ p h(j)]，g长期平均。
- **应用**：维修替换、排队动态控制（服务器分配）。
- **思路**：一分为二（现在+将来）；过去总结为状态，决策影响未来。
- **启示**：序贯决策；人生：聚焦当下（现在），考虑未来影响。五要素：状态/决策/转移/过程/目标。

## 第五章：存贮论
- **五要素**：产品（硬件）、供应（可预产）、需求、匹配（缺货可等/失）、目标（min成本）。
- **EOQ确定**：Q* = √(2KD/c_h)；总成本√(2KD c_h)；周期T* = Q*/D。
- **允许缺货**：Q* = √[2KD/c_h × (c_h+c_p)/c_p]；平衡C_h* = (c_p/(c_h+c_p))K (per cycle)。
- **非时齐/动态**：DP求订货。
- **单周期随机（Newsvendor）**：S* 满足F(S*) = (p-c)/(p-c+o-s) (连续)；E[利润] = p E[min(S,X)] - cS 等。
- **多周期**：(s,S)策略；折扣/平均用DP。
- **供应链**：牛鞭效应（信息不对称）；优化：共享信息、JIT。
- **启示**：持有/缺货/订货平衡；推广到多周期/折扣。与排队交叉：智能产品融合服务。

**总体启示**：平衡思想贯穿；从简单模型（M/M/1、EOQ）推广；应用优化设计/控制。复习重点：公式推导、思路、平衡证明。
