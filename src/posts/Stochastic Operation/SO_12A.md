# 4
## Problem
4. 某机场有4架飞机等待降落,每架飞机有不同的燃料剩余量,燃料剩余量分别为300万升、100万升、50万升和30万升。假设每架飞机降落需要100万升燃料，如果燃料不足则会坠毁。机场每次只能降落一架飞机。现在有两种策略：一种是随机选择一架飞机降落，另一种是总是选择燃料最多的飞机降落。请计算两种策略下，所有飞机都能安全降落的概率。
## Solution

**策略1: 随机选择一架飞机降落**  
在随机策略下，每次从剩余飞机中均匀随机选择一架降落。这相当于所有可能的降落顺序（4! = 24种排列）等概率发生。但在任何排列中，C和D都会在某个位置出现，且其燃料始终<100万升，导致坠毁。因此，所有飞机安全降落的概率为0。

**策略2: 总是选择燃料最多的飞机降落**  
这是一种贪婪策略，按燃料降序排列：A(300) → B(100) → C(50) → D(30)。  
- A降落：300 ≥ 100，安全。  
- B降落：100 ≥ 100，安全。  
- C降落：50 < 100，坠毁。  
- D降落：30 < 100，坠毁。  
因此，无法所有安全，概率为0。

# 5
## Problem
5. 一种新型药物对某种疾病的死亡率与血清水平Q有关，如表所示。

血清水平(万升) | 0 ≤ Q <10000 | 10000 ≤ Q <30000 | 30000 ≤ Q <50000 | Q ≥50000
---|---|---|---|---
死亡率(%) | 1.0 | 0.8 | 0.6 | 0.4

假设有5位患者，他们的血清水平分别为0,10000,30000,50000,70000。假设医生随机选择患者进行治疗，请计算所有患者都能存活的概率。
## Solution

患者血清水平：P1=0, P2=10000, P3=30000, P4=50000, P5=70000。

对应死亡率（从表）：  
- P1: 0 ≤ 0 <10000 → 死亡率1.0（存活概率=1-1.0=0）  
- P2: 10000 ≤ 10000 <30000 → 死亡率0.8（存活概率=0.2）  
- P3: 30000 ≤ 30000 <50000 → 死亡率0.6（存活概率=0.4）  
- P4: 50000 ≤ 50000（假设边界包含） → 死亡率0.4（存活概率=0.6）  
- P5: Q=70000 ≥50000 → 死亡率0.4（存活概率=0.6）

所有患者存活的概率 = ∏ (存活概率_i) = 0 × 0.2 × 0.4 × 0.6 × 0.6 = 0。

由于P1的存活概率为0，无论其他患者如何，所有存活的联合概率为0。即使考虑随机顺序（如随机排列治疗），独立性使概率不变。


# 7
## Problem
7. 假设有一个仓库有无限的容量, 需求为正态分布, 平均需求为100单位/天, 标准差为30单位/天. 假设订货提前期为5天, 补货时间为1天, 服务水平为95%. 假设仓库已经订货, 订货量为500单位, 订货时间为今天. 仓库当前库存为200单位.  计算仓库的再订货点。
## Solution

- 保护期（lead time period）= 订货提前期 + 补货时间 = 5天 + 1天 = 6天。保护期是下订单后到货物到达的时长。  
- 保护期内需求服从正态分布：平均需求 μ_L = 日平均需求 × 保护期 = 100 × 6 = 600单位。  
- 保护期内需求标准差 σ_L = 日标准差 × √保护期 = 30 × √6。  
  计算√6 ≈ 2.4495，σ_L ≈ 30 × 2.4495 ≈ 73.485单位。  
- 安全库存（safety stock）用于缓冲需求不确定性，计算为 z × σ_L，其中z是标准正态分布的反累积分布函数值，对应服务水平95%。  
  对于95%服务水平（单尾），z ≈ 1.64485（从标准正态表或Φ^{-1}(0.95)）。  
  安全库存 ≈ 1.64485 × 73.485 ≈ 120.87单位（约121单位）。  
- 再订货点 ROP = 保护期平均需求 + 安全库存 = 600 + 121 = 721单位。
