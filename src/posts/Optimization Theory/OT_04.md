# Separation Theorem

## 2.1 Theorem 2.1

In $ \mathbb{R}^{n} $ a set $ X $ is compact if and only if every sequence $ \left\{\mathrm{x}_{k}\right\} $ of points in $ X $ has a subsequence $ \left\{\mathrm{x}_{k_{j}}\right\} $ that converges to a point in $ X $.

## 2.2 Theorem 2.2

In $ \mathbb{R}^{n} $ a set $ X $ is compact if and only if $ X $ is closed and bounded.

## 2.3 Lemma 2.3

If $ X $ is a compact set in $ \mathbb{R}^{n} $ and $ f: X \rightarrow \mathbb{R} $ is a continuous function, then $ f $ attains the minimum value on $ X $, i.e there is a $ \mathbf{x}^{*} \in X $ such that $ f\left(\mathbf{x}^{*}\right)=\min _{\mathbf{x} \in X} f(\mathbf{x}) $.

## 2.3b Distance Function

Let $ d(\mathbf{y}, \mathbf{x})=\|\mathbf{y}-\mathbf{x}\| $ for any two vectors $ \mathbf{y}, \mathbf{x} $. If $ X $ is a set and $ \mathbf{y} $ is a vector, then we define the distance $ \mathbf{y} $ to $ X $ denoted by $ d(\mathbf{y}, X) $, to be
$
d(\mathbf{y}, X)=\inf \{\|\mathbf{y}-\mathbf{x}\|: \mathbf{x} \in X\}=\inf _{\mathbf{x} \in X} d(\mathbf{y}, \mathbf{x})
$

## 2.4 Lemma 2.4

- $ d(\mathbf{y}, \mathbf{x}): \mathbf{x} \in X \mapsto\|\mathbf{y}-\mathbf{x}\| $ is a continuous function on $ X $.
- If $ X $ is a nonempty closed set, then there exists a point $ \mathrm{x}^{*} \in X $ such that $ d(\mathbf{y}, X)=\left\|\mathbf{y}-\mathbf{x}^{*}\right\| $, which indicates that point $ \mathbf{x}^{*} $ in $ X $ is closest to $ \mathbf{y} $.

## 2.5 Theorem 2.5

Let $ C $ be a nonempty closed convex set in $ \mathbb{R}^{n} $ and $ \mathrm{x} \notin C $. Then there exists a nonzero vector $ \mathbf{p} $ and a scalar $ \alpha $ such that $ \mathbf{p}^{\mathbf{t}} \mathbf{x}>\alpha $ and $ \mathbf{p}^{\mathrm{t}} \mathbf{y} \leq \alpha $ for each $ \mathbf{y} \in C $.

## 2.6 Lemma 2.6

Let $ C $ be a nonempty closed convex set in $ \mathbb{R}^{n} $ and $ \mathrm{x} \notin C $. Then there exists a nonzero vector $ \mathbf{p} $ and a scalar $ \alpha $ such that $ \mathbf{p}^{\mathbf{t}} \mathbf{x}>\alpha $ and $ \mathbf{p}^{\mathrm{t}} \mathbf{y} \leq \alpha $ for each $ \mathbf{y} \in C $.

## 2.7 Corollary 2.7

Let $ C $ be a nonempty closed convex set in $ \mathbb{R}^{n} $ and $ C \neq \mathbb{R}^{n} $. Then $ C $ is the intersection of all half spaces containing $ C $.

## 2.8 Farkas' Lemma

Let A be an $ m \times n $ matrix and c be an $ n $-dimensional vector. Then exactly one of the following two systems has a solution:

System $ 1 \quad \mathbf{A} \mathbf{x} \leq \mathbf{0} $ and $ \mathbf{c}^{\mathbf{t}} \mathbf{x}>0 \quad $ for some $ \mathbf{x} \in \mathbb{R}^{n} $
System $ 2 \quad \mathbf{A}^{\mathbf{t}} \mathbf{y}=\mathbf{c} $ and $ \mathbf{y} \geq \mathbf{0} \quad $ for some $ \mathbf{y} \in \mathbb{R}^{m} $

## 2.9 Lemma 2.9

Let $ A $ be an $ m \times n $ matrix and let
$
C=\left\{\mathbf{w}: \mathbf{w}=A \mathbf{x}, \mathbf{w} \in \mathbb{R}^{m}, \mathbf{x} \in \mathbb{R}^{n}, \mathbf{x} \geq \mathbf{0}\right\} .
$

If the columns of $ A $ are linearly independent, then $ C $ is a closed convex cone.

## 2.10 Lemma 2.10

If the columns of $ A $ are not linearly independent, then any point z in $ C $ can be expressed as a nonnegative linear combination of $ p<n $ linearly independent columns of $ A $.

## 2.11 Lemma 2.11

$ C $ is a closed convex cone.

## 2.12 Corollary 2.12

Let A be an $ m \times n $ matrix and c be an $ n $-dimensional vector. Then exactly one of the following two systems has a solution:

System $ 1 \quad \mathbf{A} \mathbf{x} \leq \mathbf{0}, \mathbf{x} \geq \mathbf{0}, \mathbf{c}^{\mathbf{t}} \mathbf{x}>0 \quad $ for some $ \mathbf{x} \in \mathbb{R}^{n} $
System $ 2 \quad \mathbf{A}^{\mathbf{t}} \mathbf{y} \geq \mathbf{c}, \mathbf{y} \geq \mathbf{0} \quad $ for some $ \mathbf{y} \in \mathbb{R}^{m} $

## 2.13 Gordan's Lemma

Let A be an $ m \times n $ matrix and c be an $ n $-dimensional vector. Then exactly one of the following two systems has a solution:

$ \mathbf{A} \mathbf{x}<\mathbf{0} $  for some $ \mathbf{x} \in \mathbb{R}^{n} $

$ \mathbf{A}^{\mathbf{t}} \mathbf{y}=\mathbf{0} $ and $ \mathbf{y} \geq \mathbf{0} $  for some $ \mathbf{y} \neq \mathbf{0} \in \mathbb{R}^{m} $.

## 2.14 Theorem 2.14

Let $ S $ be a nonempty convex set in $ \mathbb{R}^{n} $ and suppose $ \overline{\mathbf{x}} \notin S $. Then there exists a hyperplane that separates $ S $ and $ \overline{\mathbf{x}} $; that is, there exists a nonzero vector $ \mathbf{p} $ in $ \mathbb{R}^{n} $ such that
$
\mathbf{p}^{t}(\mathbf{x}-\overline{\mathbf{x}}) \leq 0
$
for each $ \mathbf{x} \in \bar{S} $.

## 2.15 Lemma 2.15

Let $ S $ be a nonempty convex set in $ \mathbb{R}^{n} $ and suppose $ \overline{\mathbf{x}} \notin S $. For any finite set $ A=\left\{\mathbf{x}_{1}, \mathbf{x}_{2}, \ldots, \mathbf{x}_{n}\right\} \subseteq S $, there exists a nonzero vector $ \mathbf{p} $ in $ \mathbb{R}^{n} $ such that $ \mathbf{p}^{\mathbf{t}}(\mathbf{x}-\overline{\mathbf{x}})<0 $ for each $ \mathbf{x} \in \operatorname{co}(A) \subseteq S $.

## 2.16 Theorem 2.16

Let $ S $ be a compact set in $ \mathbb{R}^{n} $. Suppose $ \left\{C_{\alpha}\right\}_{\alpha \in A} $ is a collection of closed sets in $ S $. If for any finite set $ \left\{\alpha_{1}, \ldots, \alpha_{k}\right\}, \cap_{i=1}^{k} C_{\alpha_{i}} \neq \phi $, then $ \cap_{\alpha \in A} C_{\alpha} \neq \phi $.

## 2.17 Theorem 2.17

Let $ S_{1} $ and $ S_{2} $ be nonempty convex sets in $ \mathbb{R}^{n} $ and suppose $ S_{1} \cap S_{2}=\phi $. Then there exists a hyperplane that separates $ S_{1} $ and $ S_{2} $; that is, there exists a nonzero vector $ \mathbf{p} $ in $ \mathbb{R}^{n} $ such that
$
\inf \left\{\mathbf{p}^{\mathbf{t}} \mathbf{x}: \mathbf{x} \in \bar{S}_{1}\right\} \geq \sup \left\{\mathbf{p}^{\mathbf{t}} \mathbf{x}: \mathbf{x} \in \bar{S}_{2}\right\}
$

## 2.18 Corollary 2.18

Let $ S $ be a nonempty convex set in $ \mathbb{R}^{n} $ and suppose $ \overline{\mathrm{x}} \in \partial S $. Then there exists a hyperplane that supports $ S $ at $ \overline{\mathbf{x}} $; that is, there exists a nonzero vector $ \mathbf{p} $ in $ \mathbb{R}^{n} $ such that
$
\mathbf{p}^{t}(\mathbf{x}-\overline{\mathbf{x}}) \leq 0
$
for each $ \mathrm{x} \in \bar{S} $.

# Optimization problems

## 2.19 Definition 2.19

- Optimization Problem:
$
\begin{array}{l}
\text { minimize } f(\mathbf{x}) \\
\text { subject to } \mathbf{x} \in X
\end{array}
$

Here, $ X $ is a set in $ \mathbb{R}^{n} $ and $ f $ is a real-valued function on $ \mathbb{R}^{n} . X $ is called the feasible set and elements of $ X $ are called feasible points or feasible vectors.
- (Global) Minimizer: We say that a point $ \mathrm{x}_{0} $ in $ X $ is a (global) minimizer or that $ f $ has a (global) minimum at $ \mathbf{x}_{0} $ if $ f\left(\mathbf{x}_{0}\right) \leq f(\mathbf{x}) $ for all $ \mathbf{x} $ in $ X $.
- (Strict) Local Minimizer: We say that a point $ \mathrm{x}_{0} $ in $ S $ is a (strict) local minimizer or that $ f $ has a (strict) local minimum at $ \mathrm{x}_{0} $ if there exists a $ \delta>0 $ such that $ f\left(\mathbf{x}_{0}\right) \leq(<) f(\mathbf{x}) $ for all $ \mathbf{x}\left(\neq \mathbf{x}_{0}\right) $ in $ B\left(\mathbf{x}_{0}, \delta\right) \cap X $, where $ B\left(\mathbf{x}_{0}, \delta\right)=\left\{\mathbf{x}:\left\|\mathbf{x}-\mathbf{x}_{0}\right\|<\delta\right\} $.
- Unconstrained Problem: $ X $ is an open set in $ \mathbb{R}^{n} $
- Constrained Problem: $ X=\left\{\mathbf{x}: \mathbf{x} \in X_{0}, \mathbf{g}(\mathbf{x}) \leq \mathbf{0}, \mathbf{h}(\mathbf{x})=\mathbf{0}\right\} $, where $ f: X_{0} \rightarrow \mathbb{R}, \mathbf{g}: X_{0} \rightarrow \mathbb{R}^{m}, \mathbf{h}: X_{0} \rightarrow \mathbb{R}^{k} $.

## 2.20 Decent Direction

Let $ f $ be differentiable at $ \mathbf{x} $. If $ \nabla f(\mathbf{x}) \mathbf{d}<0, \mathbf{d} $ is denoted as a descent direction of $ f $ at x.

## 2.21 Lemma 2.21

Let $ X $ be an open set in $ \mathbb{R}^{n} $ and let $ f $ be differentiable at $ \mathbf{x} $. If there is a vector $ \mathbf{d} $ such that $ \nabla f(\mathbf{x}) \mathbf{d}<0 $, then there exists a $ \delta>0 $ such that $ f(\mathbf{x}+\lambda \mathbf{d})<f(\mathbf{x}) $ for all $ \lambda \in(0, \delta) $, so that $ \mathbf{d} $ is a descent direction of $ f $ at $ \mathbf{x} $.

## 2.22 Feasible Direction

Let $ X $ be a set in $ \mathbb{R}^{n} $ and $ \mathbf{x} \in X $. Nonzero vector $ \mathbf{d} $ is denoted as a feasible direction of $ X $ at $ \mathbf{x} $ if there is a $ \delta $ and a differentiable curve $ \boldsymbol{\xi}(t) $ such that
$
\boldsymbol{\xi}(0)=\mathbf{x}, \boldsymbol{\xi}^{\prime}(0)=\mathbf{d}, \boldsymbol{\xi}(t) \in X, t \in(0, \delta)
$
