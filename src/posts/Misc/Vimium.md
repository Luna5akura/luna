# Vimium 常用快捷键速查表

## 🎯 基本操作

| 功能               | 快捷键  |
|--------------------|---------|
| **显示所有可点击链接（Hint 模式）** | `f`     |
| 在新标签页打开链接  | `F`     |
| **向下滚动**         | `j`     |
| **向上滚动**         | `k`     |
| **向右滚动**         | `l`     |
| **向左滚动**         | `h`     |
| **滚动到页面顶部**   | `gg`    |
| **滚动到页面底部**   | `G`     |
| **向上翻页**         | `u`     |
| **向下翻页**         | `d`     |
| **返回上一个页面**   | `H`     |
| **前进到下一个页面** | `L`     |
| **重新加载页面**     | `r`     |
| **关闭当前标签页**   | `x`     |
| **恢复刚关闭的标签页** | `X`     |
| **切换到左边的标签页** | `J`     |
| **切换到右边的标签页** | `K`     |
| **打开新标签页**     | `t`     |
| **聚焦地址栏（进入输入模式）** | `o`     |
| **搜索当前页面**     | `/`     |
| **显示帮助页**       | `?`     |

## 🔧 Hint 模式（超实用）

1. **按下 `f`**，页面上所有可点击的元素（链接、按钮、输入框等）会显示提示字符（如 “AA”, “SD”等）。
2. 输入对应字母即可点击该元素，按 `F` 则会在新标签页中打开链接。

## ⚙️ 高级功能

| 功能              | 快捷键   | 说明                      |
|-------------------|----------|---------------------------|
| 打开书签或历史记录 | `b` / `B` | 与 `o` 类似                |
| 复制当前网址      | `yy`     | 类似 Vim 的 `yy` 复制行    |
| 粘贴并打开网址    | `p` / `P` | 打开剪贴板中的 URL        |
| 缩放页面          | `zi` / `zo` | 放大/缩小                  |
| 暂时禁用 Vimium（在某个页面） | `Shift + Esc` | 恢复用鼠标操作             |

## 🌟 自定义快捷键

1. 进入 Vimium 设置页面 → **Options**
2. 在 “Custom key mappings” 区域添加你自己的映射，例如：
    ```txt
    map J previousTab
    map K nextTab
    unmap f
    map s LinkHints.activate
    ```

3. 点击 “Save Changes”

## 🔄 Vimium 与 Vim 对照表

| Vim 操作 | Vimium 类似功能 |
|----------|----------------|
| `j` / `k` | 上下滚动页面 |
| `gg` / `G` | 跳到顶部 / 底部 |
| `/` 搜索  | 页面内搜索   |
| `x` 删除行 | 关闭标签页   |
| `u` 撤销   | 恢复关闭标签页（`X`） |

## 📌 小贴士

- 如果某个网页的键盘快捷键与 Vimium 冲突，可以：
  1. 进入 Vimium 设置 → **Excluded URLs and Keys**  
  2. 添加该网址，让 Vimium 在该页面禁用
- 在网页上按 `?` 可随时查看完整快捷键帮助。
